A scalable recommendation engine built using Apache Spark.

- **Algorithm:** Implemented Alternating Least Squares (ALS) for collaborative filtering.
- **Scale:** Processed the MovieLens dataset on a Hadoop cluster.
- **Accuracy:** Tuned hyperparameters to minimize Root Mean Square Error (RMSE).
