An Android application designed to assist colorblind users in navigation.

- **Accessibility:** Filters maps and UI elements to be colorblind-friendly.
- **Navigation:** Integrated Google Maps for routing and location services.
- **UX Design:** Designed high-contrast interfaces for better visibility.
