## Project Description

A collection of scripts to understand and use the LLM and prompt engineering and different parts of an AIML workflow.
The script to use local model, prompting techinques, tool use, function calling, chat templating and agentic workflow.

## Key Features

- Agentic workflow
- RAG workflow to use vectorDB
- Prompting techniques like thinking step by step, ReAct, reflction, reflexion among other.
- Graph based workflow
