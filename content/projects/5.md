## Project Description

A full stack application to contain both frontend and backend to write a full stack application to understand a full scale appliation development. Learner application that is used to learn about spring, spring boot, gradle, maven, docker, hooks, webhooks, shell scripting, jenkins, mongoDB and many other elements.

## Key Features

- Build systems (Gradle, Maven, Docker)
- Spring, spring boot
- CI/CD (Jenkins, Hooks(pre-commit, testing), Webhooks)
