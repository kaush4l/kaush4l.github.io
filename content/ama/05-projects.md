## Movie Recommender System (2016)

Built a distributed movie recommendation system using collaborative filtering on the MovieLens dataset. Implemented in both Hadoop MapReduce and Apache Spark for performance comparison.

**Technologies**: Hadoop, MapReduce, Apache Spark, Java, Python

**Technical Approach**:
- User-user similarity using modified Jaccard coefficient
- Distributed processing across cluster nodes
- Scalable architecture for large datasets
- Dual implementation for framework comparison (Hadoop vs Spark)

**Algorithms**: Collaborative filtering, distributed similarity computation, recommendation ranking

**Link**: https://github.com/kaush4l/MovieLens-Movie-Recommander-system
